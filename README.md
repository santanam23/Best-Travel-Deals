# Best Travel Deals (Front-End-Project)

As a User I want to find the best deals and discounts on hotels so I can have a more economic vacation.

I will be using Booking.com and Hotels.com to find hotel deals within the US 

I will use API's to find the best hotel deals and discounts

I will use Compare Flight Prices to ensure I get the best deal
